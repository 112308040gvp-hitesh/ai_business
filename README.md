## About Project
This project is a Business Management System developed during my PHP internship at Angel Web Solution.

I learned HTML, CSS, and Core PHP again from the basics and then worked on a live project under the guidance of my mentor. In this project I implemented database design, CRUD operations, authentication, and admin management modules.

The system allows the admin to manage employees, businesses, clients, and other business related operations.
